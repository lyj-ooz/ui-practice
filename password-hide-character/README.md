# Password Hide Character 🐵🙈

재미삼아 만들어 본 UI. 비밀번호를 한 자리 이상 입력하면 얼굴을 가린 캐릭터로 바뀌고, 비밀번호 칸이 비면 얼굴이 다시 보인다.
입력된 비밀번호 길이에 따라 캐릭터 눈동자가 따라가는(?) 그런 것도 본 적 있는데 이미지가 없어서(...) 간단하게 emoji로 구현해보았다.
